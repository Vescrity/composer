# 8-Bit 芯片音乐（Chiptune）算法作曲器 & 合成器工作站 🎹👾

这是一个专为网页端定制、视觉精美且极具物理质感的 8-bit 复古像素风算法合成器与作曲工作站。本项目不仅支持在浏览器中通过 Web Audio API 物理合成纯正的芯片音效，还内置了 **Model Context Protocol (MCP)** 客户端，允许你将算法生成的音乐事件直接同步传输到本地运行的 **Ardour DAW** 音轨中！

---

## 🎨 核心功能与视觉美学

- **吸顶固定导航顶栏**：无论怎么向下滚动浏览音轨与合成面板，顶栏的播放、暂停、循环以及全局音量面板都会牢固保持在屏幕最上方，方便随时掌控演奏。
- **CRT 复古微光扫描线**：完美的模拟街机显示屏扫描线纹理，深邃炭黑（Charcoal Gray）卡片边缘泛着幽蓝的霓虹微光，声道 VU 电平随音乐变化动态跳动。
- **经典物理纵向声道推子**：采用经典的调音台纵向音质电平推子（向上拉大音量，向下拉小音量），并配有极简的 **M**（Mute 静音）与 **S**（Solo 独奏）一键瞬态开关。
- **实时 BPM 速度对齐**：支持在乐曲循环播放时实时拖拽调节 Bpm，合成引擎会在下个小节循环开始时无缝以全新速率进行节拍重算和波形渲染。
- **模块化 Python 后端脚本**：精细重构的完整本地部署包 `python_composer/`（带 CLI 运行脚本 `chiptune_composer.py`），支持在本地终端与 Ardour 服务通信。

## 🎶 物理合成引擎架构

所有声道音效均为**网页原生端基于 Web Audio API 动态震荡器物理产生**，不使用任何录制音频或静态采样：
* **主旋律/琶音通道 (Pulse Arp)**：专门调校的超快速短促衰减（Staccato Envelope）与瞬态增益，呈现纯正、清脆活跃的 NES 经典硬件发声。
* **副旋律 (Sub/Voice)**：纯净的谐振方波，完美辅佐主旋律构架二声部对位和声。
* **低音通道 (Triangle Bass)**：利用经典的低频三角波（Triangle Wave）生成圆润且弹性十足的 8B 基音。
* **合成鼓组 (Drums - Kick, Snare, Hi-hat)**：
  * **合成底鼓 (Kick)**：采用瞬间从 150Hz 沿指数曲线跌落至 40Hz 的正弦波调频，带超短 Peak 点击音，生成弹性饱满的复古 thud 声音。
  * **合成军鼓 (Snare)**：对高斯粉色/白色噪声进行带通滤波（约 1000Hz 截止），通过 150ms 快衰减包络呈现富有颗粒感的爆破音。
  * **闭音镲click (Hihat)**：极高频带通白噪声配合超高速 60ms 衰减瞬态，输出干净利落的极高频瞬变。

## 🧠 动态算法作曲核心

支持一键编排经典日系王道和弦进行（如 **4-5-3-6 进行**）以及多种戏剧性的和声走向：
* **结构化乐段模式**：序列支持连续即兴演奏或进行经典 8 小节乐段（8-Bar Period）的严谨织体组合。
* **第 7、8 小节智能收束（Cadence）算法升级**：
  * **第 7 小节** 会进行适度的离散紧张度蓄力，以高概率主干四分拍融合极低概率的 16 分碎拍进行律动交织。
  * **第 8 小节** 则会自动挑选最和谐的经典句尾终止式之一（如全音长音、强力和声二分音符、或连续下行级进走向），并在小节的最后一个音符强力锚定到和弦的根音或五音上完成完美的情绪解脱！

## 🛠️ 项目文件树

```bash
├── package.json               # Vite + TypeScript 构建脚本
├── chiptune_composer.py       # 本地 Python CLI 作曲运行器
├── python_composer/           # 模块化 Python 音乐算法库
│   ├── __init__.py
│   ├── consts.py              # 全局调式白键音高及和弦定义
│   ├── composer.py            # 8小节乐段骨架、碎拍节奏生成器
│   ├── mcp_client.py          # Ardour MCP JSON-RPC 通信客户端
│   └── main.py                # 命令行 CLI 参数解析与轨道多进程映射
├── src/                       
│   ├── App.tsx                # 顶级页面视窗（包含固定顶栏与双栏栅格）
│   ├── types.ts               # 全局共享 TypeScript 合成声明
│   ├── composer.ts            # 前端 Web 实时算法作曲引擎
│   ├── synth.ts               # 基于 Web Audio API 构筑的合成器核心
│   └── components/            
│       ├── SettingsPanel.tsx  # 和弦走向、编排属性动态调节面板
│       ├── TrackControls.tsx  # 音轨调音台（M/S 状态灯、垂直推子）
│       ├── Visualizer.tsx     # 炫彩像素风 16步起伏走带播放网格
│       └── McpPanel.tsx       # Web 端与 Ardour 交互提示说明面板
```

## 🚀 极速起步

### 1. 开发环境
确保本地已安装运行 **Node.js 18+**。

### 2. 安装项目依赖
```bash
npm install
```

### 3. 运行本地开发服务器
```bash
npm run dev
```
打开本地控制台输出的 [http://localhost:3000](http://localhost:3000) 即可在浏览器实时进行创作和视听。

### 4. 生产打包
```bash
npm run build
```

## 🐍 本地 Python CLI 交互说明

要在本地环境与正在运行的 Ardour DAW 工作站进行同步：

```bash
# 运行 CLI 作曲服务（以种子 4536 生成 8 小节音符并在第 1 小节处灌入 Ardour 音轨）
python chiptune_composer.py --chords 4536 --start-bar 1 --length 8
```
